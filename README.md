# LTX Desktop - Product Hunt

A simple Flask web application for LTX Desktop, featured on Product Hunt.

## Features

- Home page with product information
- About page describing the product
- Contact page for feedback

## Installation

```bash
pip install -r requirements.txt
```

## Running the App

```bash
python app.py
```

Visit `http://localhost:5000` in your browser.

## Routes

- `/` - Home page
- `/about` - About LTX Desktop
- `/contact` - Contact page
